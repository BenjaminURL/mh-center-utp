# Sistema de Orientación Psicológica UTP

Credenciales de demostración:

- Estudiante: `8-111-111` / `1234`
- Psicólogo: `8-222-222` / `1234`
- Directora: `8-333-333` / `1234`

Funciones agregadas:

- Registro público exclusivo para estudiantes.
- Registro de psicólogos mediante el módulo **Nuevo colaborador** de la directora.
- El calendario estudiantil muestra únicamente psicólogos registrados por la directora.
- Consulta consolidada de reportes para la directora.

Los datos se almacenan localmente en `localStorage`; es un prototipo frontend y no una autenticación segura para producción.
