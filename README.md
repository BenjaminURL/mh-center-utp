# Sistema de Orientación Psicológica UTP

## Acceso de demostración

- Estudiante: cédula `8-111-111`, contraseña `1234`.
- Psicólogo: cédula `8-222-222`, contraseña `1234`.

El formulario acepta la cédula con o sin guiones. Estas credenciales son locales y se utilizan únicamente para la demostración del prototipo.
